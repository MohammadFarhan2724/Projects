# 🧠 Quiz App

A web-based quiz application built with HTML, CSS, and JavaScript. Covers basic web development questions.

## Features

- 8 multiple choice questions
- Instant feedback with explanations
- Progress bar
- Score summary at the end
- Restart option

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## How to Run

Just open `index.html` in any browser. No setup needed.
