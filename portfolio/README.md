# 🌐 Portfolio – Farhan Anjum

Personal portfolio website built with HTML, CSS, and JavaScript.

## Sections

- Hero with intro and CTA
- Skills
- Projects
- Experience
- Education
- Contact

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## How to Run

Just open `index.html` in any browser. No setup needed.

## Customization

- Update the contact links (email, LinkedIn, GitHub) in the Contact section
- Add your real GitHub repo links to project cards
- Add more projects as you build them
