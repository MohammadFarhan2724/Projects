# 📝 To-Do List

A simple and clean To-Do List web application built with HTML, CSS, and JavaScript.

## Features

- Add and delete tasks
- Mark tasks as complete
- Filter tasks by All / Active / Done
- Task completion counter
- Keyboard support (press Enter to add)

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## How to Run

Just open `index.html` in any browser. No setup needed.
